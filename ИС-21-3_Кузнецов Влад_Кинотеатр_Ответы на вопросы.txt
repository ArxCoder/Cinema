Вопросы:
1 - Назначение редактора Figma - создание дизайн-макета сайтов.

2 - Как скачать картинку из макета - во вкладке Layers (слои в правой части экрана)
выбрать необходимую картинку, после чего в левой части экрана в самом низу
нажать на кнопку Export для скачивания необходимой картинки.

3 - назначение вкладки inspect - блок для разработчика в котором видно настройки 
элемента и его css кода.

4 - Что такое шапка сайта и какой тег ему соответсвует - конкретно в нашей 
практической работе - это верхняя часть сайта, где присутствует иконка кинотеатра,
различного рода (пока не рабочие) ссылки, а также приветствие пользователя.
Данному объекту присущ тег <header>.

5 - Для чего необходим файл normalize.css - данный файл служит для оптимизации 
страницы сайта для различных браузеров.

6 - Ссылка для скачивания normalize.css - https://necolas.github.io/normalize.css/

7 - Команда для подключения normalize.css - <link rel="stylesheet" 
	href="css/normalize.css">

8 - Подключение шрифта Roboto - для подключения данного шрифта нужно зайти на сайт
fonts.google.com, найти там необходимый шрифт, после чего скопировать команды для 
его подключения:
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Roboto:ital,wght@0,400;1,
	700&display=swap" rel="stylesheet">
